# 安装升级 SDK

```bash
flutter channel	#查看当前渠道
flutter channel beta / flutter channel master	# 更改渠道
flutter packages get	#获取pubspec.yaml中所有的依赖关系
flutter packages upgrade	# 获取pubspec.yaml中所有列表中的依赖项的最新版本

flutter upgrade # 更新 SDK
```
